# Developer Extension Loading Guide

This Chrome Manifest V3 Extension encapsulates all Web Design & Development bookmarklets into an elegant, fast browser Action Popup.

## How to Install (Chrome, Edge, Brave, Opera)

1. **Extract ZIP File**: If downloaded as a `.zip` from our web app, unzip/extract the package into a dedicated directory (e.g., `QuickScripts-Toolbar`).
2. **Open Extensions Page**: Open your browser and navigate to the extensions control page:
   - Chrome / Edge / Brave: `chrome://extensions/`
3. **Turn on Developer Mode**: Toggle the **Developer mode** switch in the top right-hand corner of the page.
4. **Load Unpacked Directory**: 
   - Click the **Load unpacked** button.
   - Select the unzipped folder containing `manifest.json`, `popup.html`, and `popup.js`.
5. **Pin & Click**: Pin **QuickScripts Workspace** to your browser toolbar and run audits in one tap!
